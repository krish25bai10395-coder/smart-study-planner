# 📚 Smart Study Planner

A Smart Study Planner web application that helps students manage their study schedule efficiently by organizing subjects, tasks, and deadlines in a structured way.

---

## 🚀 Features

- 📅 Add and manage study tasks
- 🧠 Smart scheduling of subjects/topics
- ✅ Mark tasks as completed
- 📊 Track study progress
- 🎯 Improve productivity and time management
- 💻 Simple and user-friendly interface

---

- Frontend: HTML, CSS, JavaScript  
- Backend: Python (if used)  
- Database: (Mention if used - SQLite / JSON / None)
